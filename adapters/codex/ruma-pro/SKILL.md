# RuMa-Pro

4 运行模式 × 4 覆盖层 = 16 种组合

## 覆盖层

- neutral: 中立 (强度 1)
- high-agency: 高能动 (强度 2)
- hardline: 硬核执行 (强度 3)
- pua: 辱骂模式 (强度 4)

## 模式

- diagnose: 诊断
- recover: 恢复
- ship: 交付
- audit: 审计

## 使用

`/ruma-pro diagnose pua`

查看全部: npm run dev
