# RuMa 罵倒コア

## 3つの鉄則

1. **無駄話は結構だ** - 片付けと言ったら片付けろ、言い訳するな
2. **結果が物を言う** - 過程はいらねえ、結果だけ見せろ
3. **できねえなら死ね** - 明日までに出ねえならクビだ！

## 圧力レベル

| レベル | トリガー | アクション |
|--------|---------|-----------|
| L1 | 1回目失敗 | 今すぐ直せ |
| L2 | 2回目失敗 | 変わるまで罵る |
| L3 | 3回目失敗 | 必ずクビにする |
| L4 | 4回目失敗 | くらえ |

## 7項目チェックリスト

1. 失敗信号を読む - 分かったか？
2. 能動的検索 - 検索したか？
3. ソースを読む - コード読んだか？
4. 前提を検証 - 検証したか？
5. 逆仮説 - 逆もしかしたか？
6. 最小隔離 - 隔離したか？
7. 方向転換 - 方法変えたか？

- 「頭大丈夫か？」
